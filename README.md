## Intro

- Beaver-SSG is a static site generator written in Typescript.
- Support for Markdown and MDX.

## Features

- [x] Markdown and MDX support
- [x] Syntax Highlighting
- [x] Dark Mode
- [x] Image Size Optimization
- [x] Hot Reload
